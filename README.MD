# Wings of Darkan
Auto turn on darkan wings after changing location, entering/exiting the dungeon.
#	Крылья Даркана
Авто включение крыльев даркана после смены локации, входа / выхода из подземелья.

## Commands
Toolbox(/8) | Command description
--- | ---
**drk** | Module on/off
